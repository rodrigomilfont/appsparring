# App Sparring
[![Build Status](https://travis-ci.org/rodrigomilfont/appsparring.svg?branch=master)](https://travis-ci.org/rodrigomilfont/appsparring)
* Install via yarn:

```bash
yarn
```

* Dev:

```bash
yarn dev
```

* Prod:

```bash
yarn build
```

* Test Coverage

```bash
yarn test:coverage
```
* Test
```bash
yarn test
```
