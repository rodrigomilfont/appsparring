* Docker
	** Build image

```bash
docker build --build-arg UID=$UID -t appsparring .
```

* Install via yarn:

```bash
yarn
```

* Dev:

```bash
yarn dev
```

* Prod:

```bash
yarn build
```

* Test Coverage

```bash
yarn test:coverage
```

* Test

```bash
yarn test
```
